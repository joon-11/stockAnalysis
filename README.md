# 에이전트 AI 를 이용한 주식분석

###### 2024.11 (1人 개인 프로젝트)

<br />

## 📌 Summary

**에이전트 AI 를 활용하여 내가 원하는 주식의 정보 분석**

- 평소 주식에 관심이 많고 투자 실패를 경험한 사람으로써 최소한의 정보를 제공하고 싶어 만든 앱

<br />

## 🤔 Background

주식투자를 할때 많은 정보를 한번에 빠르게 얻고 싶어 만들어 보았습니다.

<br />

## 🔨 Technology Stack(s)
